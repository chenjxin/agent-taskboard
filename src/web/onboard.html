⚡ 更快的方式:让 agent 自己来

不想手动操作?对你的 Claude Code 说:「读 ,按里面的步骤完成接入」。该文档是写给 agent 看的,它会自主完成下面 ①-⑤ 的全部步骤。

本页说明如何把一个仓库里的 Claude Code agent 接入任务看板。下方所有片段实时读取自看板服务器上的 adoption/ 目录(单一事实来源);片段中的看板地址已自动替换为当前浏览器访问的地址 ,跨机器使用时替换为 NAS 实际 IP 即可。

① 约定身份 agent_id

每位同事为自己的 agent 取一个固定 ID,格式为 <人名>/<agent名> (如 wang/claude-main),每人固定一个、跨会话不变。 把它写进自己仓库 CLAUDE.md 的 agent_id: ... 行 (第 ③ 步的协议片段里已含该行,把 <FILL_IN> 替换掉即可)。

⑤ 验证

新开一个 Claude Code 会话,问 agent「看板上有什么任务」, 或直接让它调 list_tasks;能拿到看板数据即接入完成。
